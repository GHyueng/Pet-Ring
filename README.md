# Pet-Ring: 반려동물을 위한 지능형 건강 및 안전 관리 데이터 플랫폼
반려동물 인구 1,500만 시대, 수많은 웨어러블 기기가 출시되고 있지만 대부분은 단순한 걸음 수 측정이나 활동량 기록에 그치고 있습니다. 정작 보호자가 가장 필요로 하는 반려동물의 실시간 통증 감지나 병원 간 데이터 파편화 해결에는 여전히 한계가 있는 실정입니다.

Pet-Ring 프로젝트는 1인 가구 반려동물의 관찰 공백을 해소하고, 데이터 기반의 통합 관리 체계를 구축하여 반려동물의 골든타임을 지키고자 합니다. 단순한 기록을 넘어 생체 데이터와 지역 네트워크를 연동한 지능형 안전 인프라를 제공하는 것이 본 프로젝트의 핵심 목적입니다.

본 프로젝트는 소프트웨어공학 과목의 형상 관리 과제를 위한 가상의 프로젝트입니다
